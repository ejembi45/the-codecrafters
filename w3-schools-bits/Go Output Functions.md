## Go Output Functions
Go has three main functions for printing output to the screen, all found in the fmt package

1. fmt.Print()
Prints output on the same line, with no spaces or new line added automatically.
2. fmt.Println()
Works just like Print(), but automatically adds a new line at the end and adds spaces between values.
3. fmt.Printf()
Lets you format your output using format verbs — placeholders that get replaced with actual values.

## Some common format verbs:

    %v is used to print the value of the arguments
    %T is used to print the type of the arguments
