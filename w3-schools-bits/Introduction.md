### What Is Go

Go also known as Golang is a programming language, that is used to build a computer programes. and it was developed at google by Robert Griesemer,Rob Pike and Ken Thomson in the year 2007, it became an open source in the year 2009


##  What Is Go Used For

 * Go is used for Web development (at the server side)
 * Go is used for Cloud-native development
 * Go is used for Developing network-based programs
 * Go is used for Developing cross-platform enterprise applications


## Why We Use Go
 * Go has memory management
 * Go is fun and easy to learn
 * Go has fast run time and compilation time
 * Go works on different platforms
