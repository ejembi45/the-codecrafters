## The switch Statement
A switch statement is a cleaner and more readable alternative to writing a long chain of else if statements. Instead of checking conditions one by one, you give it a value and it jumps straight to the matching case.

## The default Case
The default case runs when none of the other cases match. Think of it like the final else in an if-else chain.

## Multiple Values in One Case
You can group multiple values into a single case using a comma.
## Switch Without an Expression
You can write a switch without a value — it then works just like an if-else chain.
