## Go Functions
A function is a block of code that performs a specific task. Instead of writing the same code over and over, you write it once inside a function and call it whenever you need it.
## Creating a Function
Use the func keyword to define a function.
## Calling a Function
To actually run the code inside a function, you call it by its name.
## Function Parameters
Parameters let you pass values into a function so it can work with them.
## Multiple Parameters
A function can accept more than one parameter.
## Return Values
Instead of printing inside the function, you can send a value back to wherever the function was called.

## Recursion Functions

Go accepts recursion functions. A function is recursive if it calls itself and reaches a stop condition.
