## Go Comments
 A comment is text that the program ignores when it runs. They're useful for explaining what your code does, making it easier to read, or temporarily disabling a block of code while you test something else. Go supports both single-line and multi-line comments.

 ## Go Single-line Comments

 Single-line comments start with two forward slashes (//)

 Any text between // and the end of the line is ignored by the compiler (will not be executed).


 ## Go Multi-line Comments

 Multi-line comments start with /* and ends with */
 Any text between /* and */ will be ignored by the compiler.
