## Go Struct
A struct is a way to group related pieces of information together under one name. Think of it like a form — a single form can hold a person's name, age, and email all in one place.

## Creating a Struct
Use the type and struct keywords to define one.

## Access Struct Members
To access any member of a structure, use the dot operator (.) between the structure variable name and the structure member
## Updating Struct Fields
You can also use the dot operator to change a field's value.
## Structs as Function Parameters
You can pass a struct into a function just like any other value.

## Nested Structs
A struct can contain another struct as a field.
